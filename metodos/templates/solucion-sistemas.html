Sistemas de Ecuaciones

Jacobi

El método de Jacobi es un algoritmo iterativo para determinar las soluciones de un sistema de ecuaciones lineales de la forma Ax = b. Es particularmente útil para matrices diagonales dominantes.

Leer más

Gauss-Seidel

El método de Gauss-Seidel es una técnica iterativa para resolver un sistema de ecuaciones lineales. Es una mejora del método de Jacobi, que generalmente converge más rápido.

Leer más

SOR

El método de Sobrerrelajación Sucesiva (SOR) es una variante del método de Gauss-Seidel que puede acelerar la convergencia mediante el uso de un factor de relajación.
