Zona de Ayuda

Bienvenido a la zona de ayuda. Aquí encontrarás información sobre cómo ingresar la información necesaria para utilizar las funciones numéricas.

Formato de Funciones (Numpy)

Las funciones se deben ingresar en formato compatible con Numpy.

Por ejemplo:
Para una función lineal: "2*x + 3"
Para una función trigonométrica: "np.sin(x) + np.cos(x)"
Para una función polinomial: "x**2 - 4*x + 3"

Formato de Matrices

Las matrices se deben ingresar separando los elementos de cada fila con comas y las filas con punto y coma.

Por ejemplo:
Para una matriz 2x2: "1,2;3,4"
Para una matriz 3x3: "1,2,3;4,5,6;7,8,9"

Formato de Tolerancia

La tolerancia se debe ingresar en forma decimal.

Por ejemplo:
Para una tolerancia de 1e-7: 0.0000001
Para una tolerancia de 1e-5: 0.00001
